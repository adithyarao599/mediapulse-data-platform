# Code of Conduct

## Our Commitment

MediaPulse Data Platform aims to provide a respectful, collaborative, and inclusive environment for anyone interested in learning, discussing, or contributing to the project.

---

## Expected Behavior

Participants are encouraged to:

* Be respectful and professional.
* Provide constructive feedback.
* Welcome different perspectives.
* Share knowledge openly.
* Focus discussions on improving the project.

---

## Unacceptable Behavior

The following behaviors are not acceptable:

* Harassment or discrimination.
* Personal attacks.
* Offensive or abusive language.
* Deliberate disruption of discussions.
* Sharing confidential or sensitive information.

---

## Scope

This Code of Conduct applies to:

* GitHub Issues
* Pull Requests
* Discussions
* Documentation contributions
* Project-related communications

---

## Enforcement

Project maintainers reserve the right to moderate discussions, remove inappropriate content, and decline contributions that do not align with this Code of Conduct.
